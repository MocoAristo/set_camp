# サービス概要
  キャンプの記録ができるサービス。
  キャンプのアイテムは種類が多く、どのアイテムをどのキャンプに
  持って行ったか分からなくなりやすい。
  キャンプごとに写真とともにアイテム・場所を記録できるサービスです。
